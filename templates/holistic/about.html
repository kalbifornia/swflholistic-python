Holistic Resources Directory

This directory (and the maps included) are here to help people find good holistic health resources in their area. The directory started out as a PDF document put together by the leader of the Weston A. Price Foundation - Naples chapter. A WAPF member, software developer Joe Kalb, thought it might be useful to have the directory accessible online with the locations mapped out. So he created this site: it is currently set up to support one map per WAPF chapter.
This directory is NOT in any way officially affiliated with the Weston A. Price Foundation. It is just something a WAPF member made to try to help out.
Feel free to check out the map for a specific WAPF chapter above, or add a resouce to the directory.
